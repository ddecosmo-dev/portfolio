Lanternfly Tracker

Timeline: September 2025 - Present
Type: Hackathon Project
Status: 2nd Place (HackCMU 17 teams)

An web application using YOLO models and data analytics to crowdsourced invasive lanternfly sightings. This project combines computer vision, data engineering, and web development to create a scalable solution for monitoring the range expansion of an invasive species.

Project Highlights

  • Placed 2nd out of 17 teams at HackCMU, demonstrating scalable prototype for invasive species monitoring
  • Built end-to-end ML pipeline from data collection through model deployment
  • Achieved 89.7% accuracy on lightweight, production-ready model
  • Implemented real-time data visualization for researchers using KDE

Project Overview

The invasive spotted lanternfly (Lycorma delicatula) harms native plantlife across the Eastern United States. Currently, there is no robust grassroots tracking of the range or concentration of these invasive insects. By utilizing a custom dataset, fine-tuning YOLO models, and developing a web facing application citings of invasive lanternflies can be logged and reported on both local and global scales.

Key Achievements

Data Collection & Augmentation
Collected 300+ high-quality real images of lanternflies in various environments, supplemented with 2,000 additional insect and environmental images from Hugging Face. This comprehensive dataset of 5,000 images was carefully curated and augmented to improve model robustness.
Model Development
Implemented transfer learning using EfficientNet-B0 with TensorFlow, achieving 89.7% classification accuracy. The lightweight model was specifically chosen for deployment at edge devices and mobile platforms, enabling real-time inference without heavy computational requirements.
Real-Time Analytics
Developed kernel density mapping visualization using SciPy and Matplotlib to transform raw sighting data into actionable insights. The system processes CSV logs in real-time, generating live concentration maps that help researchers identify hotspots and predict population expansion patterns.

Tech Stack

Python TensorFlow EfficientNet-B0 Transfer Learning SciPy Matplotlib NumPy Web Development

Results & Impact

  • Model Accuracy: 89.7% classification accuracy on diverse environmental conditions
  • Dataset: 5,000 augmented images covering various lanternfly life stages and environments
  • Deployment: Production-ready model optimized for mobile and web platforms
  • Scalability: Architecture designed to handle thousands of concurrent users submitting sightings
  • Recognition: 2nd place award among 17 competing teams at HackCMU
