Stock Sentiment Analysis Using 10-Q Forms

Timeline: October 2024 - December 2024
Type: Class Project
Peak Accuracy: 75%

A comprehensive machine learning system that predicts stock price movements following earnings announcements by analyzing sentiment in SEC 10-Q filings. This project combines natural language processing, financial data engineering, and deep learning to create an end-to-end prediction pipeline.

View on GitHub Back to Projects

Project Highlights

  • Developed dual-model architecture combining NLP (FinBERT) and traditional ML approaches
  • Built automated data pipeline processing unstructured SEC filings for 20 public companies
  • Achieved 75% classification accuracy on sentiment analysis and 70% on financial metrics
  • Analyzed which document sections contribute most to predictive accuracy

Project Overview

This project tackles the challenge of predicting stock price movements using textual and financial data from SEC 10-Q quarterly reports. By treating 10-Q filings as rich sources of company sentiment and financial health indicators, the system learns to identify predictive patterns that correlate with subsequent stock price performance.

Key Achievements

Automated Data Pipeline
Developed an end-to-end pipeline that automatically downloads 10-Q filings from the SEC's Python API and web scraping, parses unstructured financial documents, and extracts both raw text and structured financial metrics. Successfully processed data for 20 publicly traded companies with full error handling and data validation.
Dual-Model Architecture
Implemented a sophisticated two-pronged approach:
  • FinBERT Transformer: Fine-tuned on financial sentiment, analyzing textual content from 10-Q documents to capture nuanced sentiment and company-specific language patterns
  • Neural Network on Financial Metrics: Shallow network trained on quantitative indicators (EPS, ROE, etc.) to capture traditional financial health signals
Feature Engineering & Analysis
Conducted comprehensive feature analysis to determine which sections of 10-Q filings are most predictive. Results showed that combining all text features yields higher accuracy than any single section, providing valuable insights into how different parts of earnings reports correlate with stock performance.
Model Performance
Achieved peak classification accuracy of 75% for predicting binary stock price movement (up/down) post-earnings, compared to 70% baseline accuracy using only financial metrics. The FinBERT component specifically demonstrated the value of sophisticated NLP in capturing market sentiment.

Tech Stack

Python TensorFlow FinBERT Transformers SEC API Web Scraping Pandas Matplotlib NLP

Results & Analysis

  • Companies Analyzed: 20 publicly traded companies spanning multiple sectors
  • Peak Sentiment Accuracy: 75% using combined FinBERT approach
  • Financial Metrics Accuracy: 70% using traditional financial indicators
  • Feature Analysis: All document sections combined > individual sections
  • Data Volume: Thousands of 10-Q documents processed and analyzed

Key Insights

The 75% accuracy demonstrates that market-relevant sentiment is encoded in earnings reports and can be extracted through transformer models. The finding that all report sections combined yield better predictions than individual sections suggests that investors should consider holistic company narrative rather than isolated metrics. This work highlights the intersection of traditional finance and modern NLP techniques.
