Stock Sentiment Analysis Using 10-Q Forms

Timeline: October 2024 - December 2024
Type: Class Project
Peak Accuracy: 75%

A machine learning system that predicts stock price movements following earnings announcements by analyzing sentiment in SEC 10-Q filings. This project used a fine-tuned version of Fin-BERT to make effective binary predictions of stock behavior follwing a 10-Q filing.

Back to Projects

Project Highlights

  • Developed dual-model architecture combining NLP (FinBERT) and traditional ML approaches
  • Built automated data pipeline processing unstructured SEC filings for 20 public companies
  • Achieved 75% classification accuracy on sentiment analysis and 70% on financial metrics
  • Analyzed which document sections contribute most to predictive accuracy

Key Achievements

Automated Data Pipeline
Developed an end-to-end pipeline that automatically downloads 10-Q filings from the SEC's Python API and web scraping, parses unstructured financial documents, and extracts both raw text and structured financial metrics. Successfully processed data for 20 publicly traded companies with full error handling and data validation.
Dual-Model Architecture
Implemented a sophisticated two-pronged approach:
  • FinBERT Transformer: Fine-tuned on financial sentiment, analyzing textual content from 10-Q documents to capture nuanced sentiment and company-specific language patterns
  • Neural Network on Financial Metrics: Shallow network trained on quantitative indicators (EPS, ROE, etc.) to capture traditional financial health signals
Feature Engineering & Analysis
Conducted feature analysis to determine which sections of 10-Q filings are most predictive. Results showed that combining all text features yields higher accuracy than any single section.
Model Performance
Achieved peak classification accuracy of 75% for predicting binary stock price movement (up/down) post-earnings, compared to 70% baseline accuracy using only financial metrics. The FinBERT model showed improved performance over the MLP model.
