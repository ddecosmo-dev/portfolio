Thread Checker

Timeline: August 2025 - Present
Type: Personal ML Project
Accuracy: 76% Testing

A computer vision application that automatically classifies different thread types on fasteners. Initial testing was performed with three different thread types and a custom dataset of 400 images. Images were classfied using a fine-tuned MobileNetV2 model intended for edge device applications.

Project Highlights

  • Created comprehensive dataset of 400 real images, augmented to 4,000 training samples
  • Implemented data preprocessing and cleaning for standardized scaling, aspect ratio, and contrast
  • Improved classification accuracy by 15% through data augmentation
  • Deployed MobileNetV2 model to Hugging Face Space for public access

Project Overview

Thread Checker addresses the practical problem of fastener identification in manufacturing, quality control, and reverse engineering scenarios. The application uses deep learning to distinguish between three common thread types, helping engineers and technicians quickly and accurately identify fastener specifications from images alone.

Key Achievements

Dataset Creation & Augmentation
Developed a robust dataset pipeline starting with 400 high-quality real images captured under various lighting and angle conditions. Applied strategic data augmentation including rotation, scaling, brightness adjustment, and contrast normalization to expand the training set to 4,000 images without sacrificing data quality.
Data Preprocessing Pipeline
Implemented comprehensive preprocessing to standardize image scaling, maintain consistent aspect ratios, and normalize contrast across the dataset. This approach improved baseline model performance by 15% and improved generalization based on input conditions.
Model Architecture
Selected MobileNetV2 as the base architecture for its balance of accuracy and computational efficiency. Added custom pooling layers to better capture thread-specific features. The resulting model achieved 76% accuracy across all three thread types while maintaining fast inference speeds suitable for mobile and embedded deployment.
Production Deployment
Deployed the trained model to Hugging Face Spaces, creating a web interface that allows users to upload images and receive real-time classifications.
