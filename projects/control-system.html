Project Overview

This engineering project required designing a complete control system for high-temperature applications in a Department of Energy research project. The challenge involved creating tight temperature control while simultaneously developing mechanical hardware, control firmware, and monitoring software—all on an aggressive development schedule.

Key Achievements

Mechanical System Design
Engineered mechanical components and thermal management systems capable of reliably operating at high temperatures while enabling precise sensor feedback and actuator control. The design prioritized thermal stability and consistent performance across the operating range.
PID Control Implementation
Developed PID control algorithms on Arduino platform, carefully tuning parameters to achieve ±1% steady-state control tolerance. The control law was validated through extensive testing and adjusted iteratively to handle thermal dynamics and system lag.
Embedded Systems Development
Wrote efficient Arduino firmware for real-time sensor acquisition, control calculation, and communication with the monitoring system. The firmware manages multiple control loops and implements safety protocols for safe operation at temperature extremes.
Monitoring GUI Development
Built a professional C# Windows application enabling real-time visualization of system state, temperature trending, and manual control override. The GUI includes:
  • Real-time temperature display and trending graphs
  • Set-point adjustment and control mode selection
  • Historical data logging and export capabilities
  • System status indicators and safety alerts
Hardware-Software Integration
Managed seamless integration between embedded systems (Arduino), mechanical hardware, and monitoring software. Developed HTTP client communication protocols enabling reliable real-time data streaming and command execution between the control system and monitoring interface.

Technical Implementation

  • Control Architecture: Proportional-Integral-Derivative (PID) feedback control with empirical parameter tuning
  • Sensing: High-precision temperature sensors with real-time calibration and redundancy
  • Actuation: Precision heating elements with power modulation for smooth temperature transitions
  • Communication: HTTP protocol for reliable monitoring interface connectivity
  • Safety Systems: Over-temperature protection, emergency shutdown, and manual override capabilities
